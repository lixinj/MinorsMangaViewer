# MinorsMangaViewer

一款 macOS 本地漫画浏览器，用于浏览硬盘中的漫画文件，并提供版本管理功能。

## 功能

- **Finder 式图标浏览**：按作者 → 作品层级展示漫画库。
- **命名解析**：自动解析同人志和漫画文件夹的命名规则。
- **版本管理**：支持两种常见存放方式：
  - **并列版本**：同一作品的多个版本文件夹并列存放。
  - **嵌套版本**：一个主版本文件夹内包含其他版本的子文件夹（这是目前 Hamao 测试目录采用的方式）。
  浏览器会把它们合并为一个作品，阅读器顶部可切换版本。
- **阅读器**：支持单页、双页（右开 / 左开）模式，方向键 / 空格 / 点击翻页。
- **页码保留**：切换版本时尽量保持当前页码；记录每个版本的阅读进度。

## 目录结构示例

### 方式一：并列版本（推荐未来使用）

```
漫画库/
├── 南方ヒトガクシキ/
│   ├── Doujinshi/
│   │   ├── (C98) [南方ヒトガクシキ (仲村レグラ)] 作品A (艦隊これくしょん)
│   │   ├── (C98) [南方ヒトガクシキ (仲村レグラ)] 作品A (艦隊これくしょん) [無邪気漢化組]
│   │   └── (C98) [南方ヒトガクシキ (仲村レグラ)] 作品A (艦隊これくしょん) [無修正]
│   ├── Manga/
│   │   ├── (COMIC 快楽天 2022年2月号) [kakao] 作品B
│   │   └── (COMIC 快楽天 2022年2月号) [kakao] 作品B [漢化組X]
│   └── 2022/
└── 比村乳業/
    └── ...
```

### 方式二：嵌套版本（当前 Hamao 目录）

```
漫画库/
└── Hamao/
    ├── Lib - /
    │   ├── (COMIC X-EROS #43) [Hamao] 作品A [無修正][MJK-23-Z2916][無邪気漢化組]
    │   │   ├── (COMIC X-EROS #43) [Hamao] 作品A          ← 原版
    │   │   └── [Hamao] 作品A English (COMIC X-Eros #43)  ← 英译版
    │   └── ...
    ├── Manga - 2011 - 2013/
    │   └── ...
    └── (COMIC 快楽天 2022年10月号) [Hamao] 作品B [無修正]...
        └── (COMIC 快楽天 2022年10月号) [Hamao] 作品B [其他汉化组]
```

## 环境要求

- macOS 14 或更高版本
- Xcode 15 或更高版本（推荐）
- Swift 6.0 或更高版本

## 运行方式

### 方式一：命令行运行

```bash
cd /Users/minor/Downloads/MinorsMangaViewer
swift run
```

### 方式二：用 Xcode 打开

1. 双击 `Package.swift`，Xcode 会自动解析 Swift Package。
2. 选择目标 `MinorsMangaViewer` 并点击运行（⌘+R）。

## 使用说明

1. 首次启动点击工具栏「选择库目录」，选择你的漫画库根目录。
   - 如果选择的是**包含多个作者文件夹**的根目录，左侧会列出所有作者。
   - 如果直接选择**单个作者文件夹**（如 `Hamao`），应用会自动识别并把该文件夹当作唯一作者。
2. 左侧边栏会列出作者，点击作者可展开其下的各个分类/子文件夹。
3. 点击左侧某个分类后，中间网格只显示该分类下的作品。
4. 单击作品查看详情，双击打开阅读器。
5. 阅读器会**自动进入全屏**，关闭后自动恢复窗口状态。
6. 在阅读器顶部可以切换不同版本与布局，布局选择会被记住。
7. 阅读器左上角有「返回」按钮，按 ⌘W 也可以关闭阅读器并返回作品界面。
8. 如果不小心关闭了主窗口，可以按 ⌘N 或从菜单「文件 > 新建主窗口」重新打开。

## 命名规则

### 同人志

```
(集会名) [社团+作者名] 刊名 (原始IP)[标签1][标签2]…
```

例如：

```
(C98) [南方ヒトガクシキ (仲村レグラ)] 作品名 (艦隊これくしょん) [無邪気漢化組]
```

### 漫画

```
(刊号信息) [作者名] 作品名 [标签1][标签2]…
```

例如：

```
(COMIC 快楽天 2022年2月号) [kakao] 作品名 [漢化組X]
```

## 版本匹配逻辑

1. **嵌套版本优先**：如果一个作品文件夹内部包含带图片的子文件夹，这些子文件夹会被识别为同一作品的不同版本。
2. **标题合并**：在同一个作者下，标题相同（忽略空格和大小写）的作品会被合并，所有版本集中到同一个作品卡片中。
3. **标签识别**：汉化组、無修正、DL版 等信息会显示在阅读器的版本切换菜单中。

## 键盘快捷键与操作

| 快捷键 / 操作 | 功能 |
|---|---|
| ← | 上一页 |
| → | 下一页 |
| 空格 / 回车 | 下一页 |
| 点击屏幕左半部分 | 单页/双页左开：上一页；双页右开：下一页 |
| 点击屏幕右半部分 | 单页/双页左开：下一页；双页右开：上一页 |
| 右键点击图片 → 在当前页前插入空白页 | 在源文件夹中复制一张空白页到该图之前，并重新对齐双页显示 |
| 双击作品封面 | 打开阅读器 |
| ⌘W / 阅读器左上角返回按钮 | 关闭阅读器，返回作品界面 |
| ⌘N | 新建主窗口（用于主窗口关闭后重新打开） |

> 阅读器顶部的布局选择会被记住，下次打开任何作品都会沿用上次选择的布局。

## 诊断命令

如果你想知道某个目录会被如何解析，可以使用内置诊断命令（不会启动 GUI）：

```bash
swift run MinorsMangaViewer --diagnose /Users/minor/Downloads/MinorsMangaViewer/Hamao
```

或直接运行构建产物：

```bash
/Users/minor/Downloads/MinorsMangaViewer/.build/debug/MinorsMangaViewer --diagnose /Users/minor/Downloads/MinorsMangaViewer/Hamao
```
